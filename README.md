Instructions for your friends/collaborators to install and run it:
Clone the Repository:
bash


git clone https://github.com/Renukasarvade/Audio-Agent.git
cd Audio-Agent
Build the Extension:
bash


cd AudioAgent-New
npm install
npm run build
Load the Extension in Chrome:
Open Chrome and navigate to chrome://extensions/.
Enable Developer mode (toggle in the top-right corner).
Click Load unpacked (top-left corner).
Select the AudioAgent-New/dist folder.
Run the Web Portal Dashboard:
Go to the OxiqAI-MeetPortal folder.
Start a local web server (e.g., using python -m http.server 3000 or double-clicking index.html).
Open http://localhost:3000 in Chrome.
Everything is completely up-to-date and ready for your collaborative sessions! Let me know if you would like me to adjust or refine anything else!
